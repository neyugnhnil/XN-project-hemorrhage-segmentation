This repository directory was created to collaborate with team members for XN project Brain CT image hemorrhage segmentation.

It currently contains two sibling subdirectories:

- `/data` subsamples and preprocesses the source data to create a tensorflow dataset ready for keras work.
- `/tier3_neural_networks` (WIP) contains the workflow for the "Tier 3" (neural network) models. The Tier 1/2 workflows are not present in this subdirectory, but the Tier 3 workflow is meant to have access to split subset definitions from Tier 1/2.
